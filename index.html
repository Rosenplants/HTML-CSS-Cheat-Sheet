HTML Quick Resources

Tag Name Description
Needed elements for every html document
<!DOCTYPE html> HTML Declaration Tells the browser what document type to expect. At the top of every html document.
<html lang='en'> </html> html tag All html code should be nested within starting and ending html tags. The language declaration is not strictly necessary, but good practice.
<head> </head> Head Contains metadata about the html document
<body> </body> Body Defines the body of the document where all visible elements will be written.
Head Elements
<title> </title> Title The text shown in the browser's title bar or page's tab. Required element!
<style> </style> Style Used to type CSS directly in the html doc without a linked stylesheet
<link> Link Used to define a relationship between the doc and an external resource, usually the style sheet. Self-closing.
<script> </script> Script Used to write JavaScript in the document.
Text Elements
<h1> </h1>...<h6> </h6> Headers Used to create text headers in your website. h1 will be the biggest and h6 the smallest.
<p> </p> Paragraph Used to create blocks of text.
<ul> </ul> Unordered List Used to create bullet point lists with the <li> element.
<ol> </ol> Ordered List Used to create numbered lists with the <li> element.
<span> </span> Text Span Used to separate out a span of text within another element.
Media Elements
<a> </a> Anchor / Link Used to turn elements into links.
<img src="" alt="" width="" height=""> Image Used to insert images. Self-closing.
<video src="" type="" controls> </video> Video Used to insert videos. It is not a self-closing tag.
<audio src="" type="" controls> </audio> Audio Used to insert videos. It is not a self-closing tag.
Common Attributes
id="" ID Denotes unique ID of an element in a page. Used by links, Javascript, CSS and more.
class="" Class Denotes CSS class of the element.
style="" Style Applies a CSS style to an element.

CSS Quick Resources

Property Common Units or Values Description
Font Styles
font font-style font-variant font-weight font-size font-family Shorthand to set the above properties in one line. Set in the above order without commas (unless you're setting a generic font-family, the use commas there)
font-size px, em, rem, % Sets the size of the font. px sets the absolute size in pixels, the other units set the size in relation to either the parent's or root's size.
font-family FirstChoice, "Second Choice Font", generic family Specifies the font the browser should try to use and what to fall back on if that doesn't work. Use quotes around names with spaces.
font-weight normal, thin, bold, 100-900 Sets the thickness of the text characters.
font-style normal, italic, oblique, inherit, initial Makes the text italic, oblique, or normal.
font-variant small-caps Changes all lowercase letters to uppercase letters in a smaller font-size than any originally uppercase letters.
